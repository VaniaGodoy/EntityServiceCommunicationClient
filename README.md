# 📲 EntityServiceCommunicationClient

## 📌 Descrição

Aplicativo cliente responsável por consumir o serviço disponibilizado pelo projeto **EntityServiceCommunication**, utilizando **Bound Service com AIDL**.

---

## 🚀 Funcionalidades

* Conexão com serviço externo
* Chamada de métodos remotos
* Comunicação entre aplicativos Android
* Exibição de resultados na interface

---

## 🛠️ Tecnologias utilizadas

* Kotlin
* Android SDK
* Bound Service
* AIDL

---

## 🔗 Integração

Este app se comunica com o projeto:

👉 EntityServiceCommunication

⚠️ O serviço deve estar em execução antes de iniciar o cliente.

---

## ▶️ Como executar

1. Execute primeiro o app **EntityServiceCommunication**
2. Em seguida, execute este app
3. O cliente irá se conectar ao serviço automaticamente

---

## 📚 Conceitos abordados

* Bound Services
* Comunicação entre apps
* Interprocess Communication (IPC)
* Uso prático de AIDL
